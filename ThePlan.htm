Long Term Plans

Currently there are five versions specified for the Vapors engine.  All five versions are technologically feasible now, however they will entail the creation of new markets, serious support structures, and tremendous resources.  Version 3.0 is a major leap in Vapors implementation, which will include a level of realism for "Virtual Realities" that, it is planned, will create a whole new product line, market, and way of using Vapors.  Version 4.0 is likewise expected to create and open opportunities that should amaze, enable, and enrich the SuperHighway public.  Version 5.0 will step into technologies that are very cutting edge right now, however this new version should produce a level of interaction that hopefully will forever change the way we think of "Virtual Reality" and "Interactive Entertainment".

In order to bring the new Versions online, new MicroRealities products will need to be created.  These include product lines that are completely independent to gaming, and yet are necessary to the Versions 3.0 - 5.x.  They include areas in graphical and object-oriented databasing, information gathering and education, and broadband feedbacks.
